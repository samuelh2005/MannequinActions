# Mannequin Actions

Bind actions to events for your Mannequin NPCs, create intractable stories and systems. Runs entirely server-side, so your players don't need to install anything. Polymer powered.

## Setup

For setup instructions, please see the [Fabric Documentation page](https://docs.fabricmc.net/develop/getting-started/creating-a-project#setting-up) related to the IDE that you are using.
